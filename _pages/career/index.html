프로젝트

EC-CMS 2022.08

LINE의 다양한 행사를 고객에게 광고하는 방식을 관리하는 서비스입니다. 큐레이션 단위로 행사가 노출되며, 큐레이션, 행사, 행사 제공자 등 여러 데이터 간의 종속 관계 설정이 주요한 서비스입니다. 외부 회사에서 개발되었던 프로젝트를 이어받았기에 새로운 소스 코드 구조 및 처음 사용해보는 {{ site.icon.graphql }}에 적응하는 것이 관건이였던 프로젝트입니다.

담당 분야:
  • 프론트엔드 소스 코드 구조 및 서비스 구조 파악
  • 새로운 요구사항에 따른 데이터 구조 및 UI 변환
  • {{ site.icon.graphql }} 서비스 개발
Wallet Data Management (WDM) 2022.05

LINE Wallet 데이터 관리를 위한 관리자 시스템 입니다. 여러 데이터베이스 간의 데이터 이동을 서비스화 하는 것이 주 목적이며, 프론트엔드에서 데이터 유형, 쿼리, 스케줄링 방식 등에 대한 규격을 담아 서버에 요청을 보내고 백엔드에서 배치 프로세스와 젠킨스 프로그램을 통해 데이터 이동 작업을 스케줄링합니다. 데이터 사이즈와 데이터베이스의 리소스 관리 기능 또한 포함되며, 보안과 사용자별 권한이 중요한 프로젝트입니다.

담당 분야:
  • 프론트엔드 개발의 전반적인 흐름 리드
  • {{ site.icon.react }} 에서 {{ site.icon.nextjs }} 도입을 위한 리팩토링
  • 서비스의 핵심 페이지 개발
  • 데이터베이스 쿼리 및 Crontab 형식 검증 알고리즘 구현
  • Batch 작업에서 권한 코드 복호화를 위한 {{ site.icon.python }} 스크립트 작성
Monary CMS 2021.12

LINE NEWS 기사 관리를 위한 관리자 시스템 입니다. 네이버 스마트에디터 ONE (네이버 블로그 에디터와 동일한 편집도구) 를 통한 기사 등록 및 수정이 주요 기능이며, 특집 기사, 테그, 카테고리, 배너 이미지 등의 부수 기능들이 포함됩니다.

담당 분야:
  • 배너 관리 및 특집 기사 관리 페이지 개발
  • 미리보기 화면, Drag & Drop 등 복합적인 UI 기능 구현
  • 스마트에디터 부분 유지보수 담당
  • 사용자 권한에 따라 달라지는 UI 구현
