Cryptocurrency Automated Trading Simulation

Source Code: https://github.com/juhonamnam/tap-cats

{{ site.icon.python }} {{ site.icon.mysql }} {{ site.icon.telegram }} Bot API {{ site.icon.docker }}

This project aims to simulate automated trading algorithm in real time. Actual transaction does not happen, and instead, when buy or sell condition is met, real time message is sent through {{ site.icon.telegram }} Bot API. Multiple users can participate, where Telegram application works as a client in the service.
  • Simulation of automated trading algorithm through cryptocurrency exchange API
  • Implementation of WebSocket to retrieve real time price fluctuations
  • Authorization of users and admins
  • Managing user data through {{ site.icon.mysql }}
  • Telegram bot name: @tap_cats_bot
