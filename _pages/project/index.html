가상화페 자동매매 시뮬레이션

소스코드: https://github.com/juhonamnam/tap-cats

Python MySQL Telegram Bot API Docker

자동매매 알고리즘을 실시간으로 시뮬레이션 하기 위한 프로젝트입니다. 실제로 거래가 이뤄지지는 않으며, 매수 혹은 매도 조건을 충족할 경우 텔레그램 봇 API를 통해 메시지가 보내집니다. 사용자 여럿이 참여할 수 있는 구조를 갖췄으며, 텔레그램 앱이 클라이언트로 활용되는 서비스입니다.
  • 가상화폐 거래소 API를 통한 실시간 자동매매 알고리즘 시뮬레이션
  • WebSocket을 통해 실시간 가격 변동 정보 추출
  • 사용자와 관리자 권한 구현
  • MySQL을 통해 사용자 정보 관리
  • 텔레그램 봇 이름: @tap_cats_bot
