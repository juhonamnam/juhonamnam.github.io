Wallet Data Management (WDM)

소속: J&FIRST
참여 인원: 15
참여 기간: 2022.05 ~ 2023.01

{{ site.icon.react }} {{ site.icon.nextjs }} {{ site.icon.typescript }} {{ site.icon.nginx }} {{ site.icon.python }}

LINE Wallet 데이터 관리를 위한 관리자 시스템 입니다. 여러 데이터베이스 간의 데이터 이동을 서비스화 하는 것이 주 목적이며, 프론트엔드에서 데이터 유형, 쿼리, 스케줄링 방식 등에 대한 규격을 담아 서버에 요청을 보내고 백엔드에서 배치 프로세스와 젠킨스 프로그램을 통해 데이터 이동 작업을 스케줄링합니다. 데이터 사이즈와 데이터베이스의 리소스 관리 기능 또한 포함되며, 보안과 사용자별 권한이 중요한 프로젝트입니다.

담당 분야:
  • 프론트엔드 개발의 전반적인 흐름 리드
  • {{ site.icon.react }} 에서 {{ site.icon.nextjs }} 도입을 위한 리팩토링
  • 서비스의 핵심 페이지 개발
  • 데이터베이스 쿼리 및 Crontab 형식 검증 알고리즘 구현
  • Batch 작업에서 권한 코드 복호화를 위한 {{ site.icon.python }} 스크립트 작성
